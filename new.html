Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:-- 1.innerHTML - Gets the HTML content of an element,takes everything inside the element. 2.innerText - gets the text content of an element, which is visible to the user. 3.textContent - gets the text content of an element.it will return all text inside the element ,even if its hidden.

Question-3: What is event delegation in the DOM?

Answer:-- It is where a single parent element is used to handle events for many child elementa instead of adding event listeners.

Question-4: What is event bubbling in the DOM?

Answer:-- The first executed handler is on the target , then its parents and its parents,etc.
